**conway's game of life**

----------

**usage**

requires raylib to be installed in the system.

```bash
make
```

**simulation keybindings**

- hjkl - moves cursor
- space - toggles the cell marking. works only when simulation is paused
- enter - pause/resume the simulation
- left-shift + move - marks along the path of cursor movement
- right-shift + move - clears the cell along the path of cursor movement
