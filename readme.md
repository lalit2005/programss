**programss**

collection of tiny programs.  
also has the few programs that i wrote for advent of code.

- aoc-24: few questions that i solved for aoc '24 with c
- base64-zig: a base64 encoder and decoder written in zig
- sorts: an incomplete list of sorting algorithms, written in c
- http-server: a basic single threaded http server written in zig
- ropes: implementation of rope data structure in zig that self balances
- ropeviz: rope data structure visualizer, written with zig and raylib
- conway's game of life: written in c & raylib
